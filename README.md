# Smoke-Detection-System
The Smoke Detection System is designed to detect the presence of smoke and provide an immediate alert. The system uses a smoke sensor connected to an Arduino Uno. When smoke is detected above a threshold level, the buzzer and LED are activated to warn users of a potential fire hazard.

#Components
 Arduino UNO
 Buzzer 
 LCD Display
 Servo motor
 Smoke sensor

 #language
 Arduino(Embedded C)

 #Software
 Arduino IDE
